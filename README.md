# Описание

Сайт - Paint, созданный на занятиях в Кванториуме. Сайт создавался с мая по июнь
Возможности Paint-a:
- рисование
- выбор цвета кисти
- выбор толщины кисти
- выбор цвета фона
- полная очистка поля
- сохранение нарисованной картинки в виде jpg файла
- смена цвета фона независимо от рисунка
